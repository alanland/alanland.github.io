## Install Docker


## Run Rancher
```sh
$ sudo docker run -d --restart=unless-stopped -p 80:80 -p 443:443 --name rancher rancher/rancher
```

## Login

- 登录：`https://<SERVER_IP>`
- 设置 `admin` 密码
- 设置 **Rancher Server URL**

## Create the Cluster

- **Cluster** -> **Add Cluster**
- Choose **Customer**
- Enter **Cluster Name**
- Click **Next**
