## Quadrant: `工具`

## Ring: `暂缓`  2021-05-26

用来保护应用安全访问。比如在服务（比如 Nginx）前面部署一个 Pomerium，访问服务的时候先通过 Pomerium 的验证才能访问系统。
支持 Github、 Gitlab 等，相比[OAuth2-Proxy](https://oauth2-proxy.github.io/oauth2-proxy/) 
Pomerium 支持路由规则，但是使用比较复杂。
