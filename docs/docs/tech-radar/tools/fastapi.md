## Ring: `评估` 2021-04-30

一个高性能的Web框架，只支持Python3.6以上版本。性能据说可以和NodeJS、Go相比，编码简单、对API支持良好。

如果不考虑性能的优势，对OpenAPI的天然支持以及方便的容器部署是很大的优势。

官方文档很清晰，代码提示好。Flask项目可以尝试使用FastAPI。
