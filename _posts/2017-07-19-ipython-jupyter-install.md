## 安装

```shell
conda install jupyter notebook
```

或者

```shell
pip install jupyter notebook
```

## 启动

```shell
jupyter notebook
```

会自动打开浏览器,新增`notebook python3`,
对cell执行代码或者显示markdown: `shift+enter`.
