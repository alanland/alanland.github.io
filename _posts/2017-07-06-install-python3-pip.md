```shell
$ sudo apt-get install python3-pip
```
未安装成功；

使用 [get-pip.py](https://bootstrap.pypa.io/get-pip.py) 脚本:
```shell
$ sudo python3 get-pip.py
```

然后就可以使用`pip3`和`pip3.4`命令了.
